- **Запущенные задачи и legacy**: восстанавливаю поведение системы без документации — по коду, логам, базе и поведению в production. Не бросаю задачу через несколько итераций, дотягиваю до production-состояния. Пример: восстановление приёма платежей и условного формирования 54-ФЗ чеков (CloudPayments).

- **Архитектура**: проектирую от инвариантов (immutable ledger, append-only, двойная запись); закладываю транзакционность и идемпотентность с первого дня; разделяю горячий путь и тяжёлую обработку; выбираю инструмент по природе задачи. Пример: финансовое ядро на PostgreSQL; 4-уровневая система кэширования с ростом RPS в 50–100×.
- **Лидерство**: систематизирую знания в базу (основа — [infostream](https://github.com/VitalyOstanin/infostream)); налаживаю работающие процессы разработки, тестирования и деплоя; на найме отличаю реально работающих от хорошо рассказывающих.
- **AI tooling**: внедряю AI-ассистентов в работу команды; строю собственные обвязки (Claude Code plugins + публичные MCP-серверы); интегрирую с корпоративным стеком. Подход прагматичный — инструмент инженера, не замена мышления.
